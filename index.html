About Draw‑n‑Go

A simple, fun interface for turning maps into games.
Perfect for outdoor play, creative challenges, and exercising while having fun.

Get Ready

Create new session or join an existing one with your friends!

Draw

Choose a template or create your own.
Then drag and adjust to match your playground.
From template to custom in seconds!

Go!

Trace your creation with friends. Share locations, set high scores, and race against the clock for ultimate bragging rights.

Features

Draw & Go offers a comprehensive suite of features designed to enhance the experience for users, streamline session management, and make creative play easy and fun.

Game Features

Game Rooms

Join a game room by entering the session ID, or the room admin username. Collaborate with friends to complete challenges and earn rewards.

Weather Tracking

Utilizing location and real time data with weather API to provide users with accurate weather and suggestions.

Secure Authentication

User data is protected with hashed passwords stored in the database, while tokens are securely saved in cookies for session management.

Real-time location

Track the real-time location of players within the game environment, enhancing collaboration and strategy.

Admin Portal

Template Management

Create and manage the different templates for the game environment.

Users Portal

Highscore Tracking

View your high scores and achievements in the game.

Tech

Draw N Go® is powered by cutting-edge cloud and API technologies for smooth gaming experience and optimized performance

Azure SQL Database

Reliable and scalable database management for secure data storage and fast retrieval.

Azure Cloud Functions

Http triggered functions providing serverless computing for efficient execution of backend operations and business logic.

Google Maps

Integration with Google Maps for location polling and realtime tracking of multiple participants.

OpenWeather API

Provides accurate weather data based on your location, advising on optimal playing time.

Our architecture combines cloud services and real-time systems and data polling for a smooth and efficient gaming experience.
